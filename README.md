# 🕌 Islamic App

A comprehensive Kotlin-based Android application designed to provide Islamic knowledge, resources, and daily utilities for Muslims worldwide.

## 📱 About

The Islamic App is a feature-rich mobile application built with Kotlin that brings Islamic teachings, prayer times, Quranic references, and Islamic resources to your fingertips. This project aims to support Muslims in their daily spiritual journey with reliable and accessible tools.

## ✨ Features

- **Prayer Times**: Get accurate prayer times (Salah) based on your location
- **Quranic Content**: Access to Quranic verses and translations
- **Islamic Calendar**: Hijri calendar integration and Islamic dates
- **Duas & Adhkar**: Collection of daily Islamic supplications and remembrances
- **Qibla Direction**: Find the direction of the Qibla (Kaaba) from anywhere
- **Educational Resources**: Islamic knowledge and educational content
- **User-Friendly Interface**: Intuitive and clean UI design
- **Offline Support**: Access to core content without internet connection

## 🛠️ Tech Stack

- **Language**: Kotlin
- **Platform**: Android
- **Architecture**: Modern Android development practices
- **Minimum API Level**: Compatible with a wide range of Android devices

## 📋 Prerequisites

Before you begin, ensure you have the following installed:
- Android Studio (latest version)
- Kotlin plugin for Android Studio
- Java Development Kit (JDK)
- Android SDK
